# conquista-amigos

O app pode ser acessado pelo seguinte link: https://conquista-amigos-production.up.railway.app/

## Executando localmente

Use npm install para instalar as dependencias e depois rode com nodemon ou node app.js

Antes de executar o projeto:
1. remova o .example do arquivo .env.example, deixando o arquivo com nome de .env

2. Configure sua PORTA e STEAM_KEY.

Você pode conseguir sua key da api da steam aqui: https://steamcommunity.com/dev/apikey

## Configurações


Na página de configuração adicione os ids que você deseja listar na página inicial (um por linha).

**Use o steamID64!**

Exemplo:

```
700100000000
700200000000
700300000000
```

Você poderá buscar o seu steamID64 neste site: https://www.steamidfinder.com/


Na nav bar coloque o ID do jogo que deseja comparar as conquistas.
